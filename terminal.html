Biblioteca de comandos

cat - Comando com a função de executar ou criar arquivos

cd - acessa determinado dir, basta escrevermos "cd + [diretório]"

mkdir - Usado para a criação de um diretório, para utiliza-lo basta escrever o comando "mkdir + [Nome do diretório que desejar]". É muito importante ressaltar que o mesmo irá criar a pasta no local onde você está naquele momento, para saber mais sobre como mudar de local basta ver a categoria cd

rmdir - Já este utilizamos para excluir os diretórios, o funcionamento é parecido, basta escrever "rmdir + [nome do diretório que desejar excluir]".

echo - Comando que permite mostrarmos uma mensagem na tela por exemplo "echo Hello Word"

ls - Mostra as pastas e arquivos que contém no diretório selecionado
